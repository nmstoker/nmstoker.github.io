About This Website

This website lets you work on Faded Parsons Problems, a type of programming exercise where you rearrange blocks of code and fill in blanks to create a working program.

The problems hosted here are all in Python 3 and checked for correctness using doctests.


The version of the website here is a fork of this one by Pamela Fox (see GitHub repository here).

You can learn more about Faded Parsons Problems in Nate Weinman's CHI 2021 paper.
